# Breadcrumbs component

### Interfaces

#### `BreadcrumbData`

| Property | Type   | Description                           |
| -------- | ------ | ------------------------------------- |
| `name`   | String | Name of the breadcrumb item.          |
| `url`    | String | Optional URL for the breadcrumb item. |

### Properties

| Property | Type             | Description                                |
| -------- | ---------------- | ------------------------------------------ |
| `data`   | BreadcrumbData[] | Array of objects representing breadcrumbs. |
