### Properties

| Property        | Type             | Description                                    |
| --------------- | ---------------- | ---------------------------------------------- |
| `colors`        | String[]         | Array of color codes for the loader animation. |
| `[size={20}]`   | Number           | Size of the loader. Defaults to 72.            |
| `[strokeWidth]` | 1 \| 2 \| 3 \| 4 | Stroke width of the loader. Defaults to 4.     |
