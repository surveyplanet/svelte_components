## Usage

```ts
<script>
	import {avatar} from '@surveyplanet/svelte-components';
</script>
<Avatar email='ogi@sp.com' size=SIZES.SMALL imgSrc='https://surveyplanet.com'></Avatar>

```

## Parameters

| Name   | Type   | Description                                              |
| ------ | ------ | -------------------------------------------------------- |
| imgSrc | string | image source                                             |
| email  | string | email address                                            |
| size   | enum   | size of the avatar, SIZES.SMALL, SIZE.MEDIUM, SIZE.LARGE |
