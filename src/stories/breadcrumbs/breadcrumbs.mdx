## Usage

```ts
<script>
	import {Breadcrumbs} from '@surveyplanet/svelte-components';
	const props = {
		color: '#FF0000',
		size: 512,
		name: 'search',
	}
</script>

<Breadcrumbs {...props}></Breadcrumbs>
```

## Options

| Name    | Type    | Description                                                                                                                    |
| ------- | ------- | ------------------------------------------------------------------------------------------------------------------------------ |
| name='' | string  | The Breadcrumbs name.                                                                                                          |
| color   | string  | The Breadcrumbs color.                                                                                                         |
| size=24 | string  | The Breadcrumbs size in pixels. Either: 8, 16, 20, 24, 32, 48, 64, 128, 256 or 512                                             |
| debug   | boolean | Render a red backround behinde the Breadcrumbs to reveal its true edges. This can help with aliging the Breadcrumbs correctly. |
