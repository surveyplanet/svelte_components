## Usage

```ts
<script>
	import {Icon} from '@surveyplanet/svelte-components';
	const props = {
		color: '#FF0000',
		size: 512,
		name: 'search',
	}
</script>

<Icon {...props}></Icon>
```

## Options

| Name    | Type    | Description                                                                                                      |
| ------- | ------- | ---------------------------------------------------------------------------------------------------------------- |
| name='' | string  | The Icon name.                                                                                                   |
| color   | string  | The Icon color.                                                                                                  |
| size=24 | string  | The Icon size in pixels. Either: 8, 16, 20, 24, 32, 48, 64, 128, 256 or 512                                      |
| debug   | boolean | Render a red backround behinde the icon to reveal its true edges. This can help with aliging the icon correctly. |
