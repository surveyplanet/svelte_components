# Badge component

## Badge

SurveyPlanet Badge component has 4 distinct presentations. It can be Primary, Secondary, Tertiary or Disabeled.

Badge is a component that is used to trigger an action or event, such as submitting a form, opening a dialog, canceling an action, or performing a delete operation.

## Usage

```js
<script>import Badge from 'surveyplanet-svelte-Badge';</script>
```

## Badge states

There are 4 states of the Badge:

-   **Primary**
-   **Secondary**
-   **Tertiary**
-   **Disabled**

### Choosing the Badge style

The Badge style is determined by the class that is applied to the Badge. The class can be one of the following:

-   `sp-Badge--primary`
-   `sp-Badge--secondary`
-   `sp-Badge--tertiary`
-   `sp-Badge--disabled`

By adding - `sp-Badge-large` to the Badge, the Badge will be large
