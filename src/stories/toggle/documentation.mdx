## Usage

```ts
<script>
	import {Toggle} from '@surveyplanet/svelte-components';
	const props = {
		color: '#FF0000',
		size: 24,
		name: 'x',
	}
</script>

<Toggle {...props}></Toggle>
```

## Options

| Name  | Type   | Description  |
| ----- | ------ | ------------ |
| name  | string | Toggle name  |
| color | string | Toggle color |
| size  | string | Toggle size  |
