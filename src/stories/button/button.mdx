## Usage

```ts
<script>
	import {Button} from '@surveyplanet/svelte-components';
	const props = {
		label: 'Click Me!',
	}
</script>

<Button {...props}></Button>
```

## Options

| Name           | Type    | Description                                                                                                 |
| -------------- | ------- | ----------------------------------------------------------------------------------------------------------- |
| label          | string  | Button label.                                                                                               |
| disabled=false | boolean | Whether the button is disabled or not.                                                                      |
| loader=false   | boolean | Whether the button should display a loader when it is disabled.                                             |
| size='medium'  | ENUM    | The size of the button. Either: 'small', 'medium', 'large'.                                                 |
| icon           | string  | The name of the icon to use. See Iconograpy.mdx                                                             |
| mode           | ENUM    | The mode of the button. Either: 'primary', 'secondary', 'tertiary', 'danger', 'warning', 'success', 'info'. |
